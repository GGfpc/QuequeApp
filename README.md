# QuequeApp

## What is it?

QuequeApp is a Java based chat server and client inspired by WhatsApp.

## Features

  - Multiple Users
  - Individual Messages
  - Group Messages
  - Offline Mode
  - Sending Pictures
  - Message Bubbles!
  - Sent notifications
